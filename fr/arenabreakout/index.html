Dernières actualités Arena Breakout Mobile : ajustements balistiques et détection des impacts pour le tactile 2026-07-04

Arena Breakout

Tencent Games (MoreFun Studios), published globally via Level Infinite / Proxima Beta · 2022 (China) / 2023 (global)

Aussi connu sous le nom de: Arena Breakout, AB, 暗区突围, Arena Breakout Mobile

Développeur: MoreFun Studios (Tencent)Éditeur: Tencent Games (MoreFun Studios), published globally via Level Infinite / Proxima BetaSite Officiel: arenabreakout.com

Dernière vérification: 2026-07-12

Les choix S+ actuels dominent chaque classe majeure : le H416 et le FAL sont en tête des fusils d'assaut pour le jeu compétitif haut niveau, le P90 domine les SMG en tant qu'éliminateur d'escouades blindées, l'AX50 est le meilleur sniper à un coup, le Mk14 domine les DMR en tant que monstre entièrement automatique, et le lance-grenades M32 est le choix de référence pour nettoyer bâtiments et escouades. Un cran en dessous, l'AEK-971, le Groza, le MP5, le MPX, le SR-25 et le G18C complètent les options de niveau S.

Arena Breakout est un shooter d'extraction réaliste sur mobile développé par MoreFun Studios de Tencent, reconnu pour sa personnalisation d'armes poussée et authentique, inspirée du genre looter-shooter hardcore. Il est sorti en Chine continentale le 13 juillet 2022, puis mondialement (publié via Level Infinite / Proxima Beta) le 14 juillet 2023, dépassant les 10 millions de préinscriptions mondiales. Le jeu reçoit des mises à jour régulières qui rééquilibrent les accessoires, les prix et les performances des munitions sur tout son arsenal. Cette page est actualisée à chaque patch.
