也称为: Arena Breakout, AB, 暗区突围, Arena Breakout Mobile

开发商: MoreFun Studios (Tencent)发行商: Tencent Games (MoreFun Studios), published globally via Level Infinite / Proxima Beta官方网站: arenabreakout.com

最后核实时间: 2026-07-12

当前S+级武器领跑各大类别:H416与FAL在突击步枪中位居高端竞技首选;P90凭借护甲小队清除能力统治冲锋枪榜首;AX50是最强一击必杀狙击枪;Mk14作为全自动怪兽领跑DMR;M32榴弹发射器则是清理建筑与小队的首选。次一级梯队中,AEK-971、Groza、MP5、MPX、SR-25与G18C共同构成各类别的S级选择。

Arena Breakout是由腾讯MoreFun工作室开发的一款写实风格手机撤离射击游戏,以深度、真实的武器改装系统著称,灵感来自硬核搜打撤类型游戏。游戏于2022年7月13日在中国大陆上线,并于2023年7月14日通过Level Infinite / Proxima Beta在全球发行,全球预约人数超过1000万。游戏采用常态化内容更新机制,持续调整武器配件、价格与弹药性能。本页面会随每次更新刷新。
